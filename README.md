# Restaurant Delivery Application (Java)

This project is a console-based restaurant delivery system developed using Java.

## Features
- Menu management (Dishes & Drinks)
- Order processing system
- Payment handling (Cash / Card)
- Customer and Employee roles
- Delivery system based on location (Dammam, Khobar, Qatif)

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Inheritance
- Input Validation
- Modular Design
